# Parent Directory File System Management web - Application

# Overview
This is a developed web-based application demonstrating file system organization concepts through parent-child directory relationships and navigation structures.

# Features
- user login
- Teacher panel
- Admin management panel

# Technologies used
- HTML
- BOOTSTRAP
- PHP
